# 📚 TrustLens Documentation

Official documentation website for **TrustLens** — a framework-agnostic platform for **ML trustworthiness evaluation**.

TrustLens helps practitioners audit machine learning systems **beyond accuracy**, including:

- 📏 Calibration & Reliability
- ⚖️ Fairness & Bias Analysis
- 🧠 Latent Model Health
- 🚨 Failure Mode Detection
- ✅ Deployment Readiness Verdicts

This repository hosts the **generated static documentation site** for TrustLens using Sphinx-generated HTML artifacts.
